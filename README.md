# E-commerce Sales Analysis Dashboard

This project analyzes e-commerce sales data using Python and Power BI.

## Tools Used
- Python
- Pandas
- Power BI
- Data Cleaning
- Data Visualization

## Dashboard Insights
- Total Revenue
- Total Profit
- Sales by Category
- Sales by Region
- Monthly Sales Trend
- Profit Analysis
- Top 10 Customers

## Project Objective
To identify business insights and help companies make data-driven decisions.

- ecommerce-sales-dashboard.pbix → Power BI dashboard
- superstore.csv → dataset
